
# 尝试其他模型

## 说明

既然你已经使用一个训练好的回归模型构建了一个Web应用，现在请使用之前回归课程中的某个模型来重新制作这个Web应用。你可以保持原有风格，也可以重新设计，以更好地反映南瓜数据。请务必修改输入，使其与你模型的训练方法相符。

## 评分标准

| 评判标准 | 优秀                                | 合格                            | 有待改进              |
| -------- | ----------------------------------- | ------------------------------- | --------------------- |
|          | Web应用按预期运行，并已部署到云端。 | Web应用存在缺陷或出现意外结果。 | Web应用无法正常运行。 |
