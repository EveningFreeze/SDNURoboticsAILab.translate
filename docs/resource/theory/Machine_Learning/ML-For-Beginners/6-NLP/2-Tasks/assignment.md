# 让机器人回话

## 指示

在过去的几节课中，您编写了一个可以与之聊天的基本机器人程序。这个机器人会随机回答，直到您说“再见”。您能否让回答不那么随机，并在您说出特定词语（例如“为什么”或“如何”）时触发相应的回答？在扩展您的机器人时，请思考一下机器学习如何使这类工作不那么需要手动操作。您可以使用 NLTK 或 TextBlob 库来简化您的任务。
